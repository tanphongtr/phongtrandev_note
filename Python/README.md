# pip
### Uninstall all packages

```
$ pip uninstall -y -r <(pip freeze)
```


### Install a package

```
$ pip install <package-name>

$ pip install -r requirements.txt
```


# jupyter/ipython

```
$ pip install jupyter
```

Run:
```
$ ipython
or
$ jupyter notebook

```
